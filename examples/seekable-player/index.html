Piano Roll

Y = pitch, X = time. The red vertical line is the now-line. Color-coded by channel; ch 10 (drums) is grey.

Channel Notes (16ch × 128key)

Y = MIDI channel (ch 1–16), X = key number (0–127). Active notes light up in their channel color.

Waveform / Spectrum

Fill = frequency domain, line = time domain.

Mixer

S = solo (any active solo silences others), M = mute. Faders are independent of MIDI volume (CC 7 / 11).

Channel Status

Parameter reference
Prg
Program number 1–128 (General MIDI instrument)
Vol
Volume — CC 7 (0–127)
Exp
Expression — CC 11 (0–127)
Pan
Panpot — CC 10, center bar (L=0 / C=64 / R=127)
Mod
Modulation Wheel — CC 1 (0–127)
PB
Pitch Bend — center bar (−8192 to +8191)
Rev
Reverb Send — CC 91 (0–127)
Cho
Chorus Send — CC 93 (0–127)
Res
Resonance / Filter Q — CC 71 (0–127)
Brt
Brightness / Filter Cutoff — CC 74 (0–127)
Sus
Sustain Pedal — CC 64 (filled = on, threshold ≥ 64)

Bars use the channel color; brighter when notes are active.

Event Log

Incoming MIDI event stream. Toggle categories with the checkboxes; pause stops incoming events, clear discards history. Capped at 200 entries.

About

Load
File picker in Load SMF, or drag & drop a .mid file onto the Piano Roll.
Seek
Drag the seek bar or click to jump. Time and bar:beat are shown below it.
Panels
Drag a title bar to move. × hides the panel; re-open it from the top bar to reset its position.
