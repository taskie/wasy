ピアノロール

縦軸 = 音高、横軸 = 時間。赤い縦線が現在位置 (now line)。チャネルごとに色分け、ch 10 (drums) はグレー。

チャネル別ノート (16ch × 128key)

縦軸 = MIDI チャネル (ch 1〜16)、横軸 = キー番号 (0〜127)。鳴っている音はチャネル色で点灯。

波形 / スペクトル

合成出力 → AnalyserNode → destination。塗り = 周波数領域、線 = 時間領域。

このサンプルが示していること

  • smf.parseSong() でメインスレッド側からも SMF を読み、トラック数 / 解像度 / 全長を取得
  • SmfPlayer は worker 経由で再生イベントを供給するだけのパーサ。シンセは触らない
  • SynthEngine は SmfPlayer.onTimedEvent で受けた MIDI を Web Audio に流すだけ
  • 両者の橋渡しはアプリ側 (main.ts) で行う。差し替え可能性 (例: シンセを別実装に置き換え、可視化のみ SmfPlayer を使う) を確保した最小実装
